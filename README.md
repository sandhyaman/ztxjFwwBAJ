# 项目简介

## 核心功能
- 登录注册
- 数据采集
- 农产品信息展示
- 农产品数据分析与可视化
- 用户管理
- 系统管理

## 适用场景
- 农产品销售平台
- 农业电商平台
- 农产品市场分析

## 技术特点
- 采用 Flask 框架实现 Web 应用
- 使用 MySQL 数据库进行数据存储
- 前端使用 HTML、CSS 和 JavaScript 进行页面设计与交互
- 支持远程调试和稳定运行
